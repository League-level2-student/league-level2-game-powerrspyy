GLItCHeD

by: Shreyas Hosmani

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I chose to make a platformer because I wanted to make an entertaining game.

2. What I Learned

I decided to add more controls to the game to make it easier to play.

3. Architecture of My Game

The main class calls the gamepanel class to create the gamepanel and the gamepanel class calls the platform class, player class as well as the raycast and vector classes to handle collisions and other parts of the game.

4. Technical Implementation

I am proud of the improvement of the collision detection from the state at the start of the game.

5. Most Challenging Part

The most challenging part of the project is finding an efficient and easy way to implement a collision detection system.

6. Future Fixes and Updates

If I were to keep working on the game I would add different types of platforms as well as a leaderboard and a way to add custom keybinds.
